# Panduan Penggunaan Aplikasi Vitae

Panduan ini dibuat untuk membantu pengguna awam menggunakan aplikasi Vitae dari awal sampai berhasil membuat dan mengunduh CV dalam bentuk PDF.

Vitae adalah aplikasi untuk membantu karyawan menyusun CV berdasarkan data dasar dari V-People, lalu melengkapi informasi tambahan seperti foto, pengalaman kerja, pendidikan, keahlian, sertifikasi, bahasa, proyek, organisasi, dan ringkasan profil.

## 1. Tujuan Aplikasi

Aplikasi Vitae digunakan untuk:

1. Membuat draft CV karyawan secara lebih rapi.
2. Mengambil sebagian data dasar dari V-People.
3. Melengkapi data CV yang belum tersedia.
4. Melihat preview CV sebelum dicetak.
5. Mengunduh CV dalam format PDF.

## 2. Data yang Perlu Disiapkan

Sebelum membuka aplikasi, siapkan data berikut:

1. NIK Karyawan.
2. Tanggal lahir sesuai data V-People.
3. Email aktif yang bisa dibuka oleh pengguna.
4. Password baru minimal 8 karakter.
5. Nomor HP aktif.
6. Alamat domisili lengkap.
7. Tempat lahir.
8. Riwayat pengalaman kerja.
9. Riwayat pendidikan.
10. Keahlian teknis, misalnya Microsoft Excel, AutoCAD, SAP, Welding, Maintenance, dan sejenisnya.
11. Keahlian non-teknis, misalnya komunikasi, kepemimpinan, kerja sama tim, problem solving.
12. Sertifikasi atau pelatihan jika ada.
13. Bahasa yang dikuasai jika ada.
14. Proyek atau pengalaman organisasi jika ada.
15. Foto formal dalam format JPG atau PNG, maksimal 2 MB, jika ingin menambahkan foto.

| Register/Daftar | Proses membuat akun baru. |
| Field/Kolom | Kotak isian yang perlu diisi pengguna. |

## 4. Cara Membuka Aplikasi

1. Buka browser, disarankan menggunakan Google Chrome atau Microsoft Edge.
2. Ketik alamat website Vitae yang diberikan oleh admin atau HR.
3. Tekan tombol Enter pada keyboard.
4. Jika belum login, pengguna akan diarahkan ke halaman Masuk.
5. Jika sudah login, pengguna akan diarahkan ke Dashboard.

Catatan:

- Pastikan perangkat terhubung ke internet atau jaringan kantor yang diperlukan.
- Jika halaman tidak terbuka, cek kembali alamat website atau hubungi admin/HR.

## 5. Cara Membuat Akun Baru

Gunakan langkah ini jika pengguna belum memiliki akun Vitae.

1. Pada halaman Masuk, klik tombol atau link Daftar.
2. Halaman Daftar Akun Karyawan akan terbuka.
3. Isi NIK Karyawan.
4. Isi Tanggal Lahir sesuai data yang terdaftar di V-People.
5. Isi Email Login dengan email aktif.
6. Isi Password minimal 8 karakter.
7. Isi Konfirmasi Password dengan password yang sama.
8. Klik tombol Validasi dan Buat Akun.

Jika data benar, akun akan dibuat dan pengguna diarahkan ke halaman Verifikasi Email.

Jika muncul pesan gagal:

1. Periksa kembali NIK dan tanggal lahir.
2. Pastikan email ditulis dengan format benar, misalnya nama@email.com.
3. Pastikan password dan konfirmasi password sama.
4. Jika tertulis akun sudah terdaftar, gunakan halaman Masuk.
5. Jika validasi V-People sedang tidak tersedia, coba beberapa saat lagi atau hubungi admin/HR.

## 6. Cara Verifikasi Email

Setelah akun dibuat, pengguna harus mengaktifkan akun melalui email.

1. Buka email yang didaftarkan.
2. Cari email verifikasi dari aplikasi Vitae.
3. Klik link verifikasi di dalam email tersebut.
4. Setelah berhasil, pengguna akan diarahkan ke Dashboard.

Jika email tidak ditemukan:

1. Cek folder Inbox/Kotak Masuk.
2. Cek folder Spam/Junk.
3. Kembali ke halaman Verifikasi Email.
4. Klik tombol Kirim Ulang Link Verifikasi.
5. Buka kembali email dan klik link terbaru.

Catatan penting:

- Jangan membagikan link verifikasi kepada orang lain.
- Jika email yang didaftarkan salah, hubungi admin/HR untuk bantuan.

## 7. Cara Masuk ke Aplikasi

Gunakan langkah ini jika akun sudah tersedia.

1. Buka halaman Masuk.
2. Isi Email atau NIK.
3. Isi Password.
4. Jika menggunakan perangkat pribadi, pengguna boleh mencentang Ingat saya.
5. Klik tombol Masuk.

Jika login berhasil, pengguna akan masuk ke Dashboard.

Jika login gagal:

1. Pastikan Email atau NIK benar.
2. Pastikan password benar.
3. Coba ketik ulang password secara perlahan.
4. Jika masih gagal, hubungi admin/HR.

Catatan keamanan:

- Jangan centang Ingat saya jika menggunakan komputer umum atau komputer bersama.
- Jangan membagikan password kepada orang lain.

## 8. Mengenal Halaman Dashboard

Dashboard adalah halaman utama setelah pengguna berhasil masuk.

Pada Dashboard, pengguna dapat melihat:

1. Nama karyawan.
2. Jabatan atau posisi.
3. Email.
4. Nomor HP.
5. Divisi.
6. Area kerja.
7. Status CV.
8. Waktu sinkron terakhir dengan V-People.
9. Progress kelengkapan CV.

Tombol penting di Dashboard:

| Tombol | Fungsi |
| --- | --- |
| Lengkapi CV | Membuka form untuk mengisi atau mengubah data CV. |
| Lanjut Isi Draft | Melanjutkan pengisian draft CV. |
| Preview CV | Melihat tampilan CV sebelum diunduh. |
| Download PDF | Mengunduh CV dalam bentuk PDF jika data wajib sudah lengkap. |
| Keluar | Mengakhiri sesi penggunaan aplikasi. |

## 9. Cara Mengisi CV

Untuk mulai mengisi CV:

1. Dari Dashboard, klik Lengkapi CV atau Lanjut Isi Draft.
2. Halaman Lengkapi Data CV akan terbuka.
3. Isi data secara bertahap dari Step 1 sampai Step 7.
4. Gunakan tombol Berikutnya untuk pindah ke step berikutnya.
5. Gunakan tombol Sebelumnya untuk kembali ke step sebelumnya.
6. Klik Simpan Draft agar data tersimpan.
7. Klik Simpan & Preview jika ingin menyimpan dan langsung melihat tampilan CV.

Catatan:

- Data bisa disimpan walaupun belum lengkap.
- PDF hanya bisa dibuat jika data wajib sudah lengkap.
- Kolom bertanda V-People berasal dari sistem V-People dan tidak bisa diedit manual di Vitae.

## 10. Step 1 - Data Pribadi

Pada bagian Data Pribadi, pengguna melengkapi identitas utama.

Data yang muncul dari V-People:

1. Nama Lengkap.
2. NIK.
3. Tanggal Lahir.
4. Area Kerja.
5. Departemen.
6. Divisi.
7. Email.

Data yang perlu dilengkapi pengguna:

1. Foto CV, opsional.
2. Tempat Lahir.
3. Jenis Kelamin.
4. Status Pernikahan.
5. No. HP.
6. Provinsi.
7. Kabupaten/Kota.
8. Kecamatan.
9. Kelurahan/Desa.
10. Alamat Lengkap.
11. Jabatan/Posisi jika belum sesuai atau belum tersedia.

Cara memilih wilayah:

1. Pilih Provinsi terlebih dahulu.
2. Setelah itu pilih Kabupaten/Kota.
3. Setelah itu pilih Kecamatan.
4. Setelah itu pilih Kelurahan/Desa.

Jika pilihan Kabupaten/Kota, Kecamatan, atau Kelurahan/Desa belum bisa diklik, pilih data di atasnya terlebih dahulu.

## 11. Cara Menambahkan Foto CV

Foto bersifat opsional, tetapi disarankan menggunakan foto formal.

Syarat foto:

1. Format JPG atau PNG.
2. Ukuran maksimal 2 MB.
3. Wajah terlihat jelas.
4. Gunakan foto yang rapi dan profesional.

Langkah menambahkan foto:

1. Pada Step Data Pribadi, cari bagian Foto CV.
2. Klik tombol untuk memilih file foto.
3. Pilih foto dari perangkat.
4. Jika muncul halaman Crop Foto CV, atur posisi foto.
5. Gunakan Perbesar atau Perkecil jika ukuran foto belum pas.
6. Gunakan Putar Kiri atau Putar Kanan jika arah foto salah.
7. Gunakan Reset jika ingin mengulang pengaturan.
8. Klik Gunakan Foto.
9. Klik Simpan Draft agar foto tersimpan.

Jika ingin mengganti foto:

1. Upload foto baru.
2. Atur crop foto.
3. Klik Gunakan Foto.
4. Klik Simpan Draft.

Jika ingin menghapus foto:

1. Centang Hapus foto saat menyimpan.
2. Klik Simpan Draft.

## 12. Step 2 - Pengalaman Kerja

Bagian ini digunakan untuk menulis riwayat pekerjaan.

Data yang diisi:

1. Nama Posisi/Jabatan.
2. Nama Perusahaan.
3. Departemen, berasal dari V-People.
4. Divisi, berasal dari V-People.
5. Bulan mulai bekerja.
6. Bulan selesai bekerja.
7. Centang Masih bekerja sampai sekarang jika pekerjaan masih berjalan.
8. Tanggung Jawab.

Cara mengisi pengalaman kerja:

1. Isi nama posisi atau jabatan.
2. Isi nama perusahaan.
3. Pilih bulan mulai.
4. Pilih bulan selesai, jika sudah selesai.
5. Jika masih bekerja, centang Masih bekerja sampai sekarang.
6. Tulis tanggung jawab utama.
7. Gunakan daftar poin agar isi lebih mudah dibaca.

Contoh tanggung jawab yang baik:

- Melakukan pengecekan rutin mesin produksi.
- Menangani perbaikan ringan dan troubleshooting.
- Membuat laporan harian kondisi alat.
- Berkoordinasi dengan tim maintenance untuk jadwal perawatan.

Jika memiliki lebih dari satu pengalaman:

1. Klik tombol Tambah.
2. Isi pengalaman berikutnya.
3. Jika ada data yang salah atau tidak diperlukan, klik Hapus.

## 13. Step 3 - Pendidikan

Bagian ini digunakan untuk menulis riwayat pendidikan.

Data yang diisi:

1. Jenjang, misalnya SMA, SMK, D3, S1, S2.
2. Nama Institusi.
3. Jurusan.
4. Tahun Lulus.

Cara mengisi pendidikan:

1. Pilih jenjang pendidikan.
2. Isi nama sekolah, kampus, atau institusi.
3. Isi jurusan.
4. Pilih tahun lulus.
5. Klik Tambah jika ingin menambahkan pendidikan lain.
6. Klik Hapus jika ingin menghapus data yang tidak diperlukan.

Catatan:

- Pada preview CV, pendidikan dapat dibatasi maksimal 2 data dengan prioritas jenjang tertinggi.
- Pastikan pendidikan tertinggi diisi dengan benar.

## 14. Step 4 - Keahlian

Bagian ini digunakan untuk menulis kemampuan yang dimiliki pengguna.

Ada dua jenis keahlian:

1. Keahlian Teknis.
2. Keahlian Non-teknis.

Keahlian Teknis adalah kemampuan yang berkaitan langsung dengan pekerjaan atau alat kerja.

Contoh:

- Microsoft Excel
- SAP
- AutoCAD
- Welding
- Maintenance
- Electrical Troubleshooting
- Operator alat berat

Keahlian Non-teknis adalah kemampuan umum yang mendukung pekerjaan.

Contoh:

- Komunikasi
- Kerja sama tim
- Kepemimpinan
- Problem solving
- Manajemen waktu
- Disiplin

Cara mengisi:

1. Tulis keahlian pada kolom yang sesuai.
2. Pisahkan setiap keahlian dengan koma atau baris baru.
3. Gunakan istilah yang singkat dan jelas.

Contoh pengisian:

```text
Microsoft Excel, SAP, AutoCAD, Preventive Maintenance
```

## 15. Step 5 - Sertifikasi dan Pelatihan

Bagian ini bersifat opsional, tetapi sangat disarankan jika pengguna memiliki sertifikasi atau pernah mengikuti pelatihan.

Data yang diisi:

1. Nama sertifikasi atau pelatihan.
2. Penerbit/Penyelenggara.
3. Jenis, pilih Sertifikasi atau Pelatihan.
4. Tahun.
5. Berlaku sampai tahun tertentu.
6. Centang Seumur hidup / tanpa masa berlaku jika tidak ada masa berlaku.

Contoh:

- K3 Umum, Kemnaker RI, Sertifikasi, 2024.
- Basic Safety Training, Internal Training, Pelatihan, 2023.

Jika memiliki lebih dari satu sertifikasi:

1. Klik Tambah.
2. Isi data berikutnya.
3. Klik Hapus jika ingin menghapus data.

## 16. Step 6 - Tambahan Opsional

Bagian ini berisi data tambahan yang hanya tampil di CV jika diisi.

### Bahasa

Data yang diisi:

1. Bahasa.
2. Tingkat kemampuan.

Pilihan tingkat kemampuan:

1. Native.
2. Lancar.
3. Percakapan.
4. Dasar.
5. Pasif.

Contoh:

- Indonesia, Native.
- Inggris, Percakapan.

### Proyek

Data yang diisi:

1. Nama Proyek.
2. Tahun.

Contoh:

- Business Process Mapping Departemen Smelter, 2024.
- Implementasi Preventive Maintenance, 2023.

### Organisasi

Data yang diisi:

1. Nama Organisasi.
2. Jabatan/Peran.
3. Tahun Mulai.
4. Tahun Selesai.

Contoh:

- Serikat Pekerja Unit X, Ketua, 2022 sampai 2024.
- Himpunan Mahasiswa Teknik, Anggota, 2018 sampai 2020.

## 17. Step 7 - Ringkasan Profil

Ringkasan Profil adalah deskripsi singkat tentang pengalaman, keahlian, dan keunggulan pengguna.

Ketentuan:

1. Maksimal 300 karakter.
2. Ditulis singkat, jelas, dan profesional.
3. Hindari kalimat terlalu umum.
4. Sebaiknya menyebut bidang kerja, lama pengalaman, dan keahlian utama.

Pengguna bisa menulis manual atau klik tombol Generate.

Cara menggunakan Generate:

1. Pastikan data utama sudah diisi, terutama pengalaman dan keahlian.
2. Klik tombol Generate.
3. Sistem akan membuat ringkasan otomatis.
4. Baca kembali hasilnya.
5. Edit jika ada kata yang belum sesuai.
6. Klik Simpan Draft.

Contoh ringkasan:

```text
Teknisi mekanik dengan pengalaman 5 tahun di area produksi dan maintenance. Terbiasa melakukan preventive maintenance, troubleshooting, serta membuat laporan kerja harian secara disiplin.
```

## 18. Cara Menyimpan Data

Ada dua pilihan menyimpan data:

| Tombol | Fungsi |
| --- | --- |
| Simpan Draft | Menyimpan data dan tetap berada di halaman form. |
| Simpan & Preview | Menyimpan data lalu membuka halaman preview CV. |

Gunakan Simpan Draft jika:

1. Data belum lengkap.
2. Pengguna ingin melanjutkan nanti.
3. Pengguna hanya ingin menyimpan perubahan sementara.

Gunakan Simpan & Preview jika:

1. Data sudah cukup lengkap.
2. Pengguna ingin melihat tampilan CV.
3. Pengguna ingin mengecek hasil sebelum mengunduh PDF.

## 19. Data Wajib Agar PDF Bisa Dibuat

PDF hanya bisa dibuat jika data wajib berikut sudah lengkap:

1. Nama lengkap.
2. Tempat lahir.
3. Tanggal lahir.
4. Jenis kelamin.
5. Status pernikahan.
6. Alamat.
7. No. HP.
8. Email.
9. Ringkasan profil.
10. Keahlian teknis.
11. Minimal satu pengalaman kerja.
12. Minimal satu pendidikan.

Catatan:

- Nama lengkap, tanggal lahir, dan email biasanya berasal dari V-People.
- Jika data dari V-People tidak sesuai, pengguna tidak perlu mengubah langsung di Vitae. Hubungi admin/HR agar data sumber diperbaiki.
- Jika PDF gagal dibuat, aplikasi akan menampilkan daftar data yang masih kurang.

## 20. Cara Melihat Preview CV

Preview digunakan untuk mengecek tampilan akhir CV.

Langkah melihat preview:

1. Dari form CV, klik Simpan & Preview.
2. Atau dari Dashboard, klik Preview CV.
3. Periksa kembali nama, jabatan, pengalaman, pendidikan, keahlian, dan informasi lain.
4. Jika ada yang salah, klik Edit Draft.
5. Perbaiki data.
6. Simpan kembali.
7. Buka preview lagi.

Catatan:

- Data yang tampil di preview berasal dari draft terakhir yang tersimpan.
- Bagian opsional seperti sertifikasi, bahasa, proyek, dan organisasi hanya muncul jika diisi.

## 21. Cara Mengunduh CV PDF

Langkah mengunduh PDF:

1. Pastikan data wajib sudah lengkap.
2. Buka Dashboard atau halaman Preview.
3. Klik Download PDF.
4. Jika berhasil, browser akan mengunduh file CV.
5. Simpan file tersebut di folder yang mudah ditemukan.

Nama file biasanya menggunakan format:

```text
CV-nama-karyawan.pdf
```

Jika PDF tidak bisa diunduh:

1. Baca pesan error yang muncul.
2. Perhatikan daftar data wajib yang masih kurang.
3. Klik Lengkapi CV atau Edit Draft.
4. Lengkapi data yang diminta.
5. Klik Simpan Draft.
6. Coba Download PDF kembali.

## 22. Cara Keluar dari Aplikasi

Setelah selesai menggunakan aplikasi:

1. Klik tombol Keluar pada menu bagian atas.
2. Pengguna akan kembali ke halaman Masuk.

Selalu klik Keluar jika menggunakan komputer bersama.

## 23. Tips Mengisi CV agar Terlihat Profesional

1. Gunakan data yang jujur dan sesuai pengalaman.
2. Tulis pengalaman kerja dengan poin yang jelas.
3. Gunakan kata kerja aktif, misalnya mengelola, melakukan, membuat, memeriksa, menangani, mengoperasikan.
4. Hindari kalimat terlalu panjang.
5. Pastikan nomor HP aktif.
6. Gunakan email yang aktif dan profesional.
7. Isi keahlian sesuai kemampuan nyata.
8. Jangan memasukkan informasi sensitif seperti password, nomor rekening, atau data pribadi yang tidak diperlukan.
9. Periksa ulang ejaan nama perusahaan, jabatan, institusi pendidikan, dan sertifikasi.
10. Gunakan foto formal jika menambahkan foto.

## 24. Kendala Umum dan Solusinya

| Kendala | Kemungkinan Penyebab | Solusi |
| --- | --- | --- |
| Website tidak terbuka | Alamat salah, internet bermasalah, atau jaringan kantor tidak terhubung | Cek alamat website, koneksi internet, atau hubungi admin/HR. |
| Tidak bisa daftar | NIK atau tanggal lahir tidak cocok dengan V-People | Periksa kembali NIK dan tanggal lahir. Jika tetap gagal, hubungi admin/HR. |
| Email sudah digunakan | Email pernah dipakai untuk akun lain | Gunakan halaman Masuk atau hubungi admin/HR. |
| Password ditolak | Password kurang dari 8 karakter atau konfirmasi tidak sama | Gunakan minimal 8 karakter dan samakan dengan kolom konfirmasi. |
| Email verifikasi tidak masuk | Email masuk ke spam atau pengiriman tertunda | Cek spam/junk, lalu klik Kirim Ulang Link Verifikasi. |
| Tidak bisa login | Email/NIK atau password salah | Ketik ulang data login. Jika tetap gagal, hubungi admin/HR. |
| Pilihan wilayah tidak muncul | Data wilayah sebelumnya belum dipilih | Pilih Provinsi, lalu Kabupaten/Kota, lalu Kecamatan, lalu Kelurahan/Desa. |
| Foto ditolak | Format bukan JPG/PNG atau ukuran lebih dari 2 MB | Gunakan foto JPG/PNG dan kecilkan ukuran file. |
| PDF tidak bisa dibuat | Data wajib belum lengkap | Lengkapi daftar data yang muncul pada pesan error. |
| Perubahan tidak muncul di preview | Data belum disimpan | Klik Simpan Draft atau Simpan & Preview. |
| Data V-People tidak sesuai | Data sumber belum diperbarui | Hubungi admin/HR untuk koreksi data di sumber. |

## 26. Poin Penting yang Perlu Ditekankan Saat Sosialisasi

1. Akun dibuat berdasarkan validasi NIK dan tanggal lahir.
2. Email harus aktif karena diperlukan untuk verifikasi.
3. Password minimal 8 karakter dan tidak boleh dibagikan.
4. Data bertanda V-People tidak bisa diedit langsung di Vitae.
5. Pengguna boleh menyimpan draft meskipun data belum lengkap.
6. PDF hanya bisa dibuat jika data wajib lengkap.
7. Jika ada error, baca pesan yang muncul karena aplikasi memberi tahu bagian yang harus diperbaiki.
8. Jika menggunakan komputer bersama, selalu klik Keluar setelah selesai.
9. Jika lupa password atau data V-People tidak sesuai, hubungi admin/HR.

## 28. Kontak Bantuan

Jika pengguna mengalami kendala, arahkan untuk menghubungi admin atau HR dengan menyertakan informasi berikut:

1. Nama lengkap.
